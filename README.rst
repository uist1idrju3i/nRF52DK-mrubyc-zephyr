概要
********
mruby/cを用いて、Zephyrサンプルコード「Blinky」相当の動作を実現する

mruby/c取得方法
********
#. cd mrubyc
#. git submodule init
#. git submodule update

動作確認環境
********
nrf52dk_nrf52832 (PCA10040)
